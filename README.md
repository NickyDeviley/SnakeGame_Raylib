# SnakeGame C++ & Raylib

Descrição: Este projeto é uma implementação do clássico jogo "Snake", focado em lógica de programação orientada a objetos e manipulação de memória. O objetivo é controlar a cobra para comer as comidas que surgem aleatoriamente na tela, fazendo com que ela cresça, enquanto se evita colidir com o próprio corpo. O jogo resolve o desafio de gerenciamento de estados de colisão, controle de entrada de teclado em tempo real e renderização eficiente de objetos em um ambiente 2D.

Pré-requisitos: * IDE: Visual Studio 2026 (ou superior).

    Compilador: Compatível com C++17 ou superior.

    Biblioteca: Raylib (configurada no ambiente de desenvolvimento).

Instalação/Uso: 1.  Clone este repositório ou baixe os arquivos fonte.
2.  Abra o arquivo de solução (.sln) no Visual Studio 2026.
3.  Certifique-se de que a biblioteca Raylib está corretamente vinculada nas propriedades do projeto (Include e Lib paths).
4.  Compile o projeto pressionando F5 ou clicando no botão de Build.
5.  Comandos: Utilize as setas do teclado (CIMA, BAIXO, ESQUERDA, DIREITA) para controlar a cobra. O jogo encerrará automaticamente se você colidir com o próprio corpo.

Tecnologias: * C++ (Lógica principal e POO)

    Raylib (Renderização gráfica e Input)

    Visual Studio 2026 (Ambiente de desenvolvimento)

Contribuição: Outros desenvolvedores podem colaborar sugerindo melhorias no sistema de colisões, implementando um sistema de pontuação (Score), ou otimizando a geração de comida para evitar que ela apareça dentro do corpo da cobra. Sinta-se à vontade para abrir uma Issue ou enviar um Pull Request.
